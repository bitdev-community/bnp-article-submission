import { Footer } from "./footer";

A component that renders a footer component. Use with the `ThemeProvider` to enforce custom theming.

### Component usage

```js
<Footer
  leftContent={"Thanks for Visiting."}
  rightContent={"© 2022 Ober. All Rights Reserved."}
/>
```
