import { Breadcrumb } from "./breadcrumb";

A component that renders a breadcrumb to display the stream of routes.
Use with `ThemeProvider` to enforce styling.

### Component usage

It is encouraged to wrap this with a `ThemeProvider` to enforce the custom theming.

```js
<Breadcrumb
  typographyProps={{
    fontSize: 30,
  }}
  crumbs={[
    { crumb: "Home", link: "/" },
    { crumb: "Components" },
    { crumb: "Breadcrumb", link: "/component/breadcrumb" },
  ]}
/>
```
