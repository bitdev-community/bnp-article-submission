import { Typography } from "./typography";

A component that does abstracts MUI Typography. Use with `ThemeProvider` to enforce styling.

### Component usage

Use with a ThemeProvider

```js
<Typography>hello world!</Typography>
```

### Render hello world!

```js live
<Typography>hello world!</Typography>
```
